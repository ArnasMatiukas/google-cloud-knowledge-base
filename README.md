# google-cloud-knowledge-base

## Gemini CLI
Gemini CLI is an interactive command-line interface agent specializing in software engineering tasks, enabling developers to write code, execute commands, and manage projects directly from the terminal.

**Source:** [GitHub Repository](https://github.com/google-gemini/gemini-cli) | [Web Version](https://geminicli.com/)

## Agent Development Kit (ADK)
The Agent Development Kit (ADK) is a framework from Google designed to help developers build powerful multi-agent systems and accelerate the AI agent development process.

**Source:** [Official Website](https://google.github.io/adk-docs/) | [GitHub Repository](https://github.com/google/adk-python) | [ADK Samples](https://github.com/google/adk-samples)

## Community Gemini Quick Starts

### Skills.sh Gallery
A community-driven directory showcasing skills, prompts, and configurations designed specifically for extending and optimizing the Gemini CLI experience.

**Source:** [https://skills.sh](https://skills.sh)

### Antigravity Kit
A highly popular template toolkit for AI agents that provides ready-to-use skills, agents, and workflows for quick project bootstrapping.

**Source:** [https://github.com/vudovn/antigravity-kit](https://github.com/vudovn/antigravity-kit)

### Gemini Kit
A comprehensive extension kit tailored for the Gemini CLI featuring numerous pre-configured AI agents and commands for automated planning, testing, and security reviews.

**Source:** [https://github.com/nth5693/gemini-kit](https://github.com/nth5693/gemini-kit)

## Google Workspace Integrations

### Gemini CLI Workspace Extension
A dedicated extension for the Gemini CLI that leverages a local MCP server, allowing AI agents to securely query and interact with personal Google Workspace data like emails and calendars via natural language.

**Source:** [https://github.com/gemini-cli-extensions/workspace](https://github.com/gemini-cli-extensions/workspace)

## Google Cloud Local MCP Servers

### Cloud Run MCP
An MCP server designed to enable AI agents to deploy and manage applications on Google Cloud Run.

**Source:** [https://github.com/GoogleCloudPlatform/cloud-run-mcp](https://github.com/GoogleCloudPlatform/cloud-run-mcp)

### GKE MCP
An MCP server that allows compatible AI agents to interact with and manage Google Kubernetes Engine (GKE) clusters.

**Source:** [https://github.com/GoogleCloudPlatform/gke-mcp](https://github.com/GoogleCloudPlatform/gke-mcp)

### GenAI Toolbox
An open-source MCP server that provides tools for AI agents to connect to, query, and interact with various databases.

**Source:** [https://github.com/googleapis/genai-toolbox](https://github.com/googleapis/genai-toolbox)

### gcloud MCP
An MCP server that exposes the functionality of the Google Cloud CLI (gcloud), allowing AI agents to manage Google Cloud resources directly.

**Source:** [https://github.com/googleapis/gcloud-mcp](https://github.com/googleapis/gcloud-mcp)

## Google Cloud Remote MCP Servers
Google Cloud provides hosted MCP servers that allow AI agents to securely interact with cloud resources without needing to run a local server instance. Supported products include:

- AlloyDB for PostgreSQL
- BigQuery
- Bigtable
- Cloud Run
- Customer Experience Agent Studio
- Cloud SQL
- Cloud Logging
- Cloud Monitoring
- Compute Engine
- Firestore
- GKE
- Google Security Operations
- Managed Service for Apache Kafka
- Resource Manager
- Spanner
- Vertex AI Search
- Developer Knowledge API
- Maps Grounding Lite

**Source:** [https://docs.cloud.google.com/mcp/supported-products](https://docs.cloud.google.com/mcp/supported-products)
